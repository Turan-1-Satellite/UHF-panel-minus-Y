# Turan-1 UHF Panel −Y in KiCad

## About

This repository contains the KiCad 9.0 project files for the **TURAN-1 UHF Panel −Y PCB** — a directional UHF-band antenna panel mounted on the (−Y) face of the Turan-1 CubeSat, providing UHF communications coverage (400–450 MHz) in the boresight direction of that panel.

---

## Requirements

- KiCad 9.0 or later

---

## First Set Up

All required custom libraries and 3D models are included as a git submodule. To clone the repository together with custom libraries, you need to recursively clone it:

```bash
git clone --recurse-submodules https://github.com/Turan-1-Satellite/UHF-panel-minus-Y.git
```

---

## Open the Project

Open the `UHF-panel-minus-Y.kicad_pro` file from the project root directory.
